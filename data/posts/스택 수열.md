https://www.acmicpc.net/problem/1874

임의의 수열을 스택으로 구현할 수 있는지 없는지
push는 "+", pop은 "-"

for문 안에도 data = int(input())를 입력할 수 있음

```
n = int(input())
count = 1
stack = []
result = []

for i in range(1, n + 1):
    data = int(input())

    while count <= data:
        stack.append(count)
        count += 1
        result.append('+')

    if stack[-1] == data:
        stack.pop()
        result.append('-')
    else:
        print('NO')
        exit(0)

print('\n'.join(result))

```
