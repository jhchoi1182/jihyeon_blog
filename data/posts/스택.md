https://www.acmicpc.net/problem/10828

파이썬으로 스택 구현

```
n = int(input())
stack = []  # 스택(stack) 초기화
for _ in range(n):
    command = input().strip().split(' ')
    if command[0] == 'push':
        stack.append(int(command[1]))
    elif command[0] == 'pop':
        if len(stack) == 0:
            print(-1)
        else:
            print(stack.pop())
    elif command[0] == 'size':
        print(len(stack))
    elif command[0] == 'empty':
        if len(stack) == 0:
            print(1)
        else:
            print(0)
    elif command[0] == 'top':
        if len(stack) == 0:
            print(-1)
        else:
            print(stack[-1])

```
