친구 네트워크에 몇 명이 있는지 구하는 문제

합집합(union-find) 찾기 알고리즘

```
def find(x):
    if x == parent[x]:
        return x
    else:
        p = find(parent[x])
        parent[x] = p
        return parent[x]

def union(x, y):
    x = find(x)
    y = find(y)
    if x != y:
        parent[y] = x

parent = [i for i in range(5)]

union(1, 4)
union(2, 4)

for i in range(1, len(parent)):
    print(find(i), end=' ')

```
