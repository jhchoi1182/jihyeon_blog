배열의 오른쪽 큰 수 구하기

중첩 for문으로 구현하는 것을 스택을 이용해 n으로 시간복잡도를 개선할 수 있음

```
n = int(input())
arr = list(map(int, input().split()))
stack = []
NGE = [-1] * n

for i in range(n):
    x = arr[i]
    if len(stack) == 0 or stack[-1][0] >= x:
        stack.append((x, i))
    else:
        while len(stack) > 0:
            previous, index = stack.pop()
            if previous >= x:
                stack.append((previous, index))
                break
            else:
                NGE[index] = x
        stack.append((x, i))
for x in NGE:
    print(x, end=' ')
```
