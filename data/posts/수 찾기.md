https://www.acmicpc.net/problem/1920

N개의 정수 중 특정 값이 있는지 찾는 문제

int(input()) 사용이랑 답 모양 맞추는게 제일 어려운 듯...

set 자료형 사용

```
n = int(input())
array = set(map(int, input().split()))
m = int(input())
x = list(map(int, input().split()))

for i in x:
    if i not in array:
        print('0')
    else:
        print('1')
```
